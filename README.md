# EditKaro.in – Video Editing & Marketing Agency Website 🎮🚀

[![Live Demo](https://img.shields.io/badge/Live-Demo-green)](https://web-project-xnt2.onrender.com/)
[![GitHub](https://img.shields.io/badge/Source-Code-blue)](https://github.com/Sunil93021/EditKaro-Web-Project)

EditKaro.in is a modern and responsive portfolio website built for a creative video editing and marketing agency. It showcases service categories like gaming edits, eCommerce ads, football edits, documentary-style content, and more. The site is designed with engaging visuals, smooth navigation, and user-friendly responsiveness across devices.

---

## 🌐 Live Website

👉 [Visit the Website](https://web-project-xnt2.onrender.com/)

---

## 📌 Features

- 🎮 Service categories with video previews
- 🧠 Informative FAQ and Services page
- 📈 Portfolio page with organized sections
- 💬 Contact form integrated with Google Sheets using Node.js
- 📱 Fully responsive for mobile and desktop
- ✨ Smooth animations and clean UI

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js (used for contact form)
- **Database:** Google Sheets (via Google Apps Script)
- **Hosting:** Render.com

## 🚀 Setup Instructions

1. **Clone the Repository**

```bash
git clone https://github.com/Sunil93021/EditKaro-Web-Project.git
cd EditKaro-Web-Project
```

2. **Install Node.js Dependencies** (only if backend is used for contact form)

```bash
npm install
node server/index.js
```

3. **Open `index.html`** in any browser or host using [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in VS Code.

## 📈 Form Submission (Google Sheets Integration)

This website uses Google Sheets as a backend to collect data from the contact form. When a user submits the form, their input is sent to the following Google Sheet via Google Apps Script:

📝 View Form Responses: [Google Sheet](https://docs.google.com/spreadsheets/d/1iOkJ1DIv8wqF3maCVK6DNrJy3pYN0YzBXbSrosEnJ2w/edit?usp=drivesdk)

> Note: Ensure that the Google Apps Script linked to the form handles POST requests and appends the data to the correct sheet columns.
